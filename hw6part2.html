Homework #6, Part 2 - Use String objects to search text, and create a new Window object to present message

This script determines the number of occurences of a single letter (case insensitive) in a text block.
If the letter is found, the number of occurences is displayed on this page.
If the letter is not found, a new window is created to display the result message.
